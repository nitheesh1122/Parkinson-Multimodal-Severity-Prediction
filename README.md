# Explainable Multimodal Framework for Parkinson Disease Detection and Severity Prediction

## Project Overview

This project aims to develop an explainable multimodal machine learning framework for Parkinson disease detection and severity prediction using:

* Speech Features
* Handwriting Features
* Age
* Gender

The proposed framework integrates multiple biomarkers to improve diagnostic accuracy and provide interpretable predictions using SHAP Explainable AI.

## Problem Statement

Existing Parkinson disease prediction models rely primarily on speech-based biomarkers and binary classification. These approaches do not fully capture the clinical manifestations of Parkinson disease.

## Proposed Solution

A multimodal framework combining:

* Speech Analysis
* Handwriting Analysis
* Demographic Features
* Severity Prediction
* Explainable AI

## Expected Outputs

* Parkinson Detection
* Severity Classification
* Feature Importance Analysis
* SHAP Explainability Report
